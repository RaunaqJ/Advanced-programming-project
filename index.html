FILM CINEMAX

A Professional Desktop Application for Managing Your Film Library

PROJECT OVERVIEW

Film Cinemax is a comprehensive desktop application built with PyQt6 and Flask that provides a seamless solution for managing and organizing your personal film library. The application combines a modern, intuitive graphical interface with a robust backend API to deliver a professional film management experience. With 50 carefully curated films across multiple categories, users can search, filter, sort, and manage their collection with ease.

üîç Search Films

Type a film name in the "Film Name" search box and click "Search" to instantly find movies matching your query. The library will filter and display only the films that match your search term.

üé≠ Filter by Category

Use the "Category" dropdown to filter films by type: Drama, Crime, Action, Sci-Fi, Romance, or Animation. Select "All" to view your entire collection. Changes are applied instantly.

üìä Sort Films

Click the "Sort" button to access sorting options. Choose to sort by Year (Newest), Runtime (Longest), or Name (A-Z). Your library will reorganize instantly.

üóëÔ∏è Delete Films

Select a film from the table and click "Delete" to remove it from your library. A confirmation dialog will appear before permanent deletion to prevent accidental removal.

üöÄ Quick Start Guide

  1. Launch the application - it will automatically load 50 films from the database
  2. Browse films using the table or search by name using the Search feature
  3. Filter films by category using the Category dropdown (Drama, Action, etc.)
  4. Click any film to see its detailed description and runtime information
  5. Sort your collection by clicking the Sort button and choosing your preference
  6. Add new films by clicking "Add Film" and filling the dialog with film details
  7. Delete unwanted films by selecting them and clicking the Delete button
  8. All changes are saved automatically to the film database
